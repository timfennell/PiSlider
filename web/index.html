Sony WiFi Camera

Tap a network to select it:
Tap Scan to find the camera network, or Already Connected if wlan1 is already joined.
  • Camera: Network → Ctrl w/ Smartphone → ON
  • Camera: Network → Send to Computer → ON (for capture)
  • Password is shown on the camera screen under Network settings

Sony USB Tether

Connect USB-C cable, then tap Detect Camera.
  • Camera: Setup → USB Connection → PC Remote
  • Camera: Setup → USB Power Supply → OFF
  • RAW files downloaded directly to Pi SSD
  • ✅ Holy Grail available — full exposure control via gphoto2
  • ~50–150ms trigger latency (faster than WiFi)

Sony S2 Cable Mode

  • Shutter fired via GPIO optocoupler → S2 cable (~5ms latency)
  • Exposure managed entirely on the camera body
  • Files saved to the camera's SD card — no download to Pi
  • ⚠ Holy Grail unavailable in S2 mode (requires WiFi API)
  • Ideal for motion-detect timelapse with Pi Camera as detector
~1.2A run current — for steep rail angles / heavy loads
